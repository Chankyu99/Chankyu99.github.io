## 3.1 데이터 유형

**변수(variable)** : 공통의 측정 방법으로 얻은 같은 성질의 값

> 통계학에서 변수의 개수를 **'차원'** 이라고도 한다.

변수유형은 크게 2가지로 구분된다.

1. **<mark>양적변수(수치형 변수)</mark>** : 숫자로 나타낼 수 있는 변수
    - 이산형(countable) 변수 : 얻을 수 있는 값이 점점이 있는 변수
    - 연속형(measurable) 변수 : 간격없이 이어지는 값

2. **<mark> 질적변수(범주형 변수)</mark>** : 숫자가 아닌 범주로 나타낼 수 있는 변수
    - 명목형(nominal) 변수 : 범주 이름이 의미 없는 변수
    - 순서형(ordinal) 변수 : 범주 이름이 의미 있는 변수

## 3.3 통계량

통계량은 수집한 데이터로 계산해 얻은 값으로, 몇 가지 통계량을 계산하여 요약하면, 데이터가 어떻게 분포하는지 정략적으로 특징을 지을 수 있다.

데이터 그 자체의 성질을 기술하고 요약하는 통계량을 **<mark>기술통계량</mark>** 또는 **<mark>요약통계량</mark>**이라고 한다.

다양한 기술통계량 중 가장 자주 쓰이는 용어들을 정리한다면 다음과 같다.

* **<mark>대푯값(representative value)</mark>** : <U>대략적인 분포 위치</U>, 즉 대표적인 값을 정량화하기 위해 사용하는 통계량
    - 평균값(mean) : $$\bar{x} = \frac{1}{n} \sum_{i=1}^{n} x_i$$
    - 중앙값(median) : 크기 순으로 값을 정렬했을 때 중앙에 위치한 값
    - 최빈값(mode) : 데이터 중 가장 자주 나타나는 값

> 1. 분포가 좌우 대칭일수록 3가지 값은 대체로 일치
> 2. 평균값은 이상치(Outlier)에 매우 민감

* **<mark>산포도(Dispersion)</mark>** : <U>데이터의 분포의 퍼짐 정도</U>, 즉 데이터가 얼마나 퍼져있는지 정량화하기 위해 사용하는 통계량
    - 분산(variance) : $$s^2 = \frac{1}{n} \sum_{i=1}^{n} (x_i - \bar{x})^2$$
    - 표준편차(standard deviation) : $$s = \sqrt{s^2} = \sqrt{\frac{1}{n} \sum_{i=1}^{n} (x_i - \bar{x})^2}$$

> 1. $s^2 \geq 0$
> 2. 모든 값이 같다면 $s^2 = 0$
> 3. 산포도가 클수록 $s^2$가 커진다.

## 3.4 확률
> "발생여부가 불확실한 사건의 발생 가능성을 숫자로 표현"

### 3.4.1 확률변수가 1개일 경우

<U>어떤 사건 A가 발생할 확률</U>을 $P(A)$라고 하면

$$0 \leq P(A) \leq 1 $$

**<mark>확률변수</mark>** : 사건 $A$와 같이 확률이 달라지는 변수

**<mark>실현값</mark>** : 사건 $A$가 실제로 취하는 값

**<mark>확률분포</mark>** : 가로축에 확률변수, 세로축에 해당 확률변수의 발생가능성을 표시한 분포

**<mark>확률밀도함수</mark>** : 연속형 확률변수의 확률분포에서 일정범위를 두고 확률을 구하는 함수

$$P(a \leq X \leq b) = \int_{a}^{b} f(x) dx$$ 

![확률밀도함수](/assets/img/probability_density_function.png)

**<mark>기댓값</mark>** : 변수가 확률적으로 얼마나 발생하기 쉬운가를 평균적인 값으로 표현

- 이산형 : $E(X) = \sum_{i=1}^{n} x_i P(x_i)$
- 연속형 : $E(X) = \int_{a}^{b} x f(x) dx$

**<mark>분산</mark>** 과 **<mark>표준편차</mark>** : 기댓값에서 얼만큼 떨어졌는가?

- 이산형 : $V(X) = \sum_{i=1}^{n} (x_i - E(X))^2 P(x_i)$
- 연속형 : $V(X) = \int_{a}^{b} (x - E(X))^2 f(x) dx$

> 특징 
> 1. $V(X) \geq 0$
> 2. 모든 값이 같다면 $V(X) = 0$
> 3. 기댓값에서 떨어진 값이 많을 수록(산포도가 클수록) $V(X)$가 커진다.

**<mark>왜도</mark>** : 분포가 좌우대칭에서 어느정도 벗어났는가? 

**<mark>첨도</mark>** : 분포가 얼마나 뾰족한가? / 그래프 꼬리가 차지하는 비율이 얼마인가?

### 3.4.2 확률변수가 2개일 경우

**<mark>동시확률분포</mark>** : 사건 $X$, 사건 $Y$사이의 관계성을 파악하기 위한 확률분포

**<mark>독립</mark>** : 한 사건이 발생할 확률이 다른 사건 확률에 영향을 주지 않는 경우 $P(X,Y) = P(X) \times P(Y)$

**<mark>조건부확률</mark>** : 하나의 확률변수의 정보를 알 때 다른 확률변수의 확률 $P(X \| Y) = \frac{P(X \cap Y)}{P(Y)}$

> 이때, $X$,$Y$가 **독립**일경우 $P(X \| Y) = P(X)$

## 3.5 이론적인 확률 분포

* **<mark>정규분포(normal distribution)</mark>** : 가우스 분포(Gaussian distribution)라고도 한다. 

$$f(x) = \frac{1}{\sqrt{2\pi\sigma^2}} e^{-\frac{(x-\mu)^2}{2\sigma^2}}$$

> $\mu$ : 평균값, $\sigma$ : 표준편차, 이 2가지 모수(parameter)로 결정한다. <br>
> $N(\mu, \sigma)$으로 나타내고, 여기서 평균과 표준편차가 0,1인 분포를 **<mark>표준정규분포</mark>**라고 한다.

![정규분포](/assets/img/normal_distribution.png)
_각 퍼센테이지는 하위%확률을 의미 (예 : 68%는 평균값에서 1 표준편차 이내의 확률)_

**<mark>표준화(Z-score)</mark>** : 정규분포의 모수를 표준정규분포의 모수로 변환하는 과정

$$Z = \frac{X - \mu}{\sigma}$$

> Z-score를 통해 본래 점수가 아닌 분포 내 위치로 평가 가능
